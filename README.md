# YES Base

YES base 是一套可以配置的管理系统前端开发框架

#### 安装

``` 
bower install yes-base
```

#### 技术构架

YES base 基于 Angularjs 技术

#### 结构目录

``` 
├── Base
│   ├── css
│   │   ├── main.css
│   │   ├── vendor.css
│   ├── scripts

│   │   ├── yes.app.js
│   │   ├── yes.app.min.js
│   └── vendor
│       ├── ace
│       ├── angular-schema-form
│       ├── angular-ui-grid
│       ├── ie8
```
